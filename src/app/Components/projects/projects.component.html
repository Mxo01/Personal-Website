LoveJob

LoveJob is a Web App written in Angular and ASP.NET Core with C#. It's main goal is to simplify job seeking for people who's searching for a job. The application is designed for two types of users, job seekers and those who work for a company (and thus upload open or available job positions). The Front-End part is completly written in Angular with TypeScript (with Bootstrap and CSS for the styles) and the Back-End part is written in C# usign ASP.NET Core as a framework to use Models & Controllers and it's connected with Microsoft SQL Server to query the data from Database.

MyLibretto

MyLibretto is a Progressive Web App written in React. It's a University project that allows users to keep track of their career progress. They can then upload grades, fees, share notes (photos, files, videos), and look at their achievements through easy-to-understand statistics and graphs. The Front-End is written in React with JavaScript (using only CSS for the styles) and the Back-End is written using Firebase (BaaS).

Crafty

CRAFTY is a University project totally written in Python. The project works with a Dataset comprising Bitcoin transactions, specifically those included in blocks between 2009 and 2012. Data Analysis/Mining is the part where I perform statistical analysis on the data and generating graphical visualizations and then I implement a clustering algorithm. Finally I use Web Scraping to de-anonymize as many clusters as possible utilizing libraries such as Beautiful Soup and Selenium to create bots that automate the search process.
